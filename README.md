## Настройка проекта
```
npm install
```

### Компиляция и горячая перезагрузка для разработки
```
npm run serve
```

### Компиляция и минимизация для продакшена
```
npm run build
```

### Линтинг и исправление файлов
```
npm run lint
```

См. [Справочник по конфигурации](https://cli.vuejs.org/config/).
